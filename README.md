# 250410_r3f

ICS MEDIAの記事『』のサンプルコードです。

パッケージマネージャーを使用しています。以下のコマンドを行うことでモジュールが追加されます。

```bash
npm install
```

起動は、以下のコマンドを行うか、package.jsonのscriptsのdevコマンドを実行することで開発画面が起動できます。

```bash
npm run vite
```
